# AI Betting Bot - Android App

Flutter-based Android application for AI-powered football betting predictions.

## Features

- AI-powered match analysis
- Real-time match data
- Multiple leagues support
- User authentication
- Demo mode for testing

## Build

APK is automatically built via GitHub Actions on every push.

Download from: **Actions → Build Android APK → Artifacts**

## Tech Stack

- Flutter 3.16
- Riverpod (State Management)
- Dio (HTTP Client)
- Go Router (Navigation)
